# Tunisian Bank Card Number Validation Problem

Contexte : Implémentation de l'algorithme de Luhn  pour la validation de cartes bancaires tunisiennes.

Éléments techniques inclus :

Manipulation de tableaux (déclaration, remplissage, parcours)

Algorithmes de traitement de chaînes

Validation et contrôle de saisie

Calculs arithmétiques modulaires

Ce projet constitue un bon exercice pour travailler la logique algorithmique et la programmation structurée autour des tableaux et des opérations numériques.
