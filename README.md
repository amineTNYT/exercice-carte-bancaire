# carte-bancaire
Contexte : Implémentation de l'algorithme de Luhn modifié avec inversion pour la validation de cartes bancaires tunisiennes.  Éléments techniques inclus :  Manipulation de tableaux (déclaration, remplissage, parcours)  Algorithmes de traitement de chaînes  Validation et contrôle de saisie  Calculs arithmétiques modulaires
